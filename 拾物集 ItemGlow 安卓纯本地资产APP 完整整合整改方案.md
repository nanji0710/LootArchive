# 拾物集 ItemGlow 安卓纯本地资产APP 完整整合整改方案

## 一、项目基础信息

1. **APP 名称**
中文名：拾物集 \| 英文名：ItemGlow

2. **核心定位**
纯本地私有物品资产管理工具，无联网、不上传数据，适配安卓 10 \(API29\) 及以上，自用收藏 / 数码 / 衣物 / 工具台账；
视觉风格：苹果 iOS 通透毛玻璃（Blur）设计，支持浅色 / 深色自动切换、自定义首页背景图。

3. **开发技术栈 Jetpack Compose**

    - 数据库：Room \(SQLite 本地加密存储\)

    - UI：Material3 \+ Accompanist Blur（玻璃模糊核心）

    - 图表：Compose Charts（饼图 / 柱状图）

    - 图片：Coil 本地图片压缩、相册 / 相机调取

    - Excel 导入导出：EasyExcel 轻量化安卓适配版

    - 动效：Compose 原生 Animation \+ Lottie 空状态动画

    - 版本管理：Git \+ GitHub 私有仓库，VSCode 开发
